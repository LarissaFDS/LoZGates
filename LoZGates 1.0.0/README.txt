LoZ Gates - Gerador de Circuito Lógico

📌 Sobre o Projeto

LoZ Gates é uma aplicação interativa para trabalhar com expressões lógicas, gerando circuitos equivalentes e facilitando a visualização das operações através de um interface gráfica.

🎯 Funcionalidades

Conversão de Expressões Lógicas: Interpretação de sentenças em lógica proposicional.

Tabela Verdade: Geração automática da tabela verdade correspondente.

Mapa de Karnaugh: Análise e simplificação de expressões lógicas.

Comparação de Expressões: Determinação de equivalência lógica entre sentenças.

Geração de Circuito Lógico: Representação gráfica interativa do circuito correspondente.

🚀 Como Executar

1️⃣ Pré-requisitos

Certifique-se de ter instalado:

Python 3.8+

Pip

2️⃣ Instalação das Dependências

Execute no terminal:

pip install customtkinter pygame pillow

3️⃣ Execução do Programa

No diretório do projeto, execute:

python front.py

📂 Estrutura do Projeto

LoZ_Gates/
│-- equivalencia.py  # Comparação de expressões lógicas
│-- front.py         # Interface gráfica
│-- circuito_logico.py  # Geração de circuitos com pygame
│-- karnaugh.py      # Geração de Mapas de Karnaugh
│-- converter.py     # Conversão para Álgebra Booleana
│-- assets/          # Ícones e imagens do projeto

🖥️ Tecnologias Utilizadas

Python 🐍

CustomTkinter (Interface gráfica)

Pygame (Visualização do circuito lógico)

Pillow (Manipulação de imagens)
